Pomoc - Panel zarządzania GlobalVista

Struktura panelu

Panel zarządzania GlobalVista pozwala na wprowadzanie danych do różnych modułów systemu analitycznego. Wybierz odpowiednią sekcję z menu po lewej stronie.

SQL Server

W tej sekcji możesz wprowadzać dane strukturalne do relacyjnej bazy danych MS SQL Server. Dostępne moduły:

  • Moduł Sprzedażowy - zarządzanie produktami, klientami, zamówieniami
  • Moduł Bankowy - zarządzanie kontami i transakcjami bankowymi
  • Moduł Geograficzny - zarządzanie krajami i regionami
  • Sprzedawcy - zarządzanie profilami sprzedawców

MongoDB

Ta sekcja pozwala na zarządzanie elastycznymi strukturami danych w MongoDB. Dostępne kolekcje:

  • Historie Zakupów - kompleksowe dane o historii zakupów klientów
  • Logi Transakcji - szczegółowe logi transakcji bankowych
  • Zachowania Klientów - analiza aktywności i zachowań klientów
  • Profile Sprzedawców - rozszerzone profile sprzedawców z danymi sprzedażowymi

Azure

W tej sekcji możesz zarządzać integracjami z usługami chmurowymi Azure:

  • Azure Storage - zarządzanie przechowywaniem danych
  • Data Factory - konfiguracja przepływów przetwarzania danych
